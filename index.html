Admin Generator Advanced is a plugin ment to add and modify the admin  WordPress panel. At least that’s the long term vision for it. the 0.8 Version Allows you to add infinite and configurable metaboxes to posts and pages. Each metabox can contain infinite textboxes, textareas, checkboxes or custom select (dropdown boxes).

Everything is configurable trough a admin panel user interface with just the point of a mouse and a keyboard. Just click and write text. Everything automatigally appears in the admin panel once it’s saved.
